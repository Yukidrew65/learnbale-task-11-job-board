# 📋 Job Board App

A premium, modern Job Board application tailored for finding localized, verified jobs (Cleaning, Care, and Factory roles) across Lagos. Built using React, Vite, and TypeScript with a clean, Apple-inspired responsive user interface.

---

## ✨ Features

- **Apple-Inspired Design**: Soft gradients, interactive hover states, dynamic animations, and micro-interactions.
- **Categorized Job Listings**: Effortlessly browse verified jobs across key categories:
  - 🧼 **Cleaning** (Office cleaners, housekeepers, industrial spaces)
  - 🩺 **Care** (Home care assistant, elderly care, childcare)
  - 🏭 **Factory** (Production operators, machine operators, warehouse pickers)
- **Fluid Segmented Controls**: Quick-filtering tabs that instantly filter listings with interactive indicators.
- **Detailed Accordions**: Expand and collapse individual listings to read deep-dive specifications (wages, schedules, exact locations, and full descriptions) without losing your scroll context.
- **Responsive Layout**: Designed for visual excellence on both mobile screens and desktop monitors.

---

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vite.dev/)
- **Programming Language**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Modern, modular inline styling with custom CSS keyframes for buttery smooth rendering.

## 🚀 Getting Started

Follow these steps to run the application locally on your machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v18 or higher is recommended).

### Installation

1. Clone or download this repository.
2. Open your terminal in the project directory.
3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server

To launch the local development server with hot-module replacement (HMR):
```bash
npm run dev
```
The application will run on [http://localhost:5173/](http://localhost:5173/).

### Production Build

To compile a highly optimized production bundle:
```bash
npm run build
```
The built files will be output to the `dist/` directory, ready to be deployed to static hosting platforms like Netlify or Vercel.
